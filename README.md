<h1>📊 Regional Sales Analysis</h1>

<h2>📌 Project Overview</h2>
<p>
This project analyzes regional sales data to identify performance trends,
highlight underperforming regions, and generate actionable business insights.
The analysis includes data cleaning, feature engineering, and exploratory data analysis (EDA).
</p>

<hr>

<h2>🎯 Business Objective</h2>
<ul>
    <li>Analyze sales performance across regions</li>
    <li>Identify high-performing and low-performing regions</li>
    <li>Understand sales trends and distribution</li>
    <li>Support data-driven business decisions</li>
</ul>

<hr>

<h2>🗂️ Dataset Description</h2>
<p>The dataset contains transactional sales data with attributes such as:</p>
<ul>
    <li>Region</li>
    <li>Sales</li>
    <li>Quantity</li>
    <li>Profit</li>
    <li>Product / Category (if applicable)</li>
    <li>Time-based fields</li>
</ul>
<p>
Raw data was cleaned and transformed before analysis.
</p>
