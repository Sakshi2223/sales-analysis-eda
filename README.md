\# 📊 Sales Analysis Using Exploratory Data Analysis (EDA)



\## 📌 Project Overview

This project focuses on analyzing regional sales data to uncover trends, patterns, and insights that support business decision-making.



\## 🛠 Tools \& Technologies

\- Excel

\- SQL

\- Power BI

\- Python (Pandas, Matplotlib, Seaborn)

\- Jupyter Notebook



\## 📂 Project Structure

| **data/** | 📄 Raw sales dataset |
| **notebooks/** | 📓 Jupyter notebooks for EDA |
| **screenshots/** | 📊 Power BI dashboard visuals |


\## 🔍 Key Analysis Areas

\- Sales performance by region

\- Customer and product-level insights

\- Data cleaning and transformation

\- Exploratory visual analysis
